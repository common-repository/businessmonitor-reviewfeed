Met de BusinessMonitor Reviewfeed toon je de door ons verzamelde reviewscores en reviews op je website.

== Description ==

Met behulp van de standaard Reviewfeed Plugin wordt de feedback van je klanten op een survey in de vorm van een aanbeveling of review direct doorgeplaatst naar je website. BusinessMonitor is specialist in online onderzoek en heeft uitgebreide expertise in het realiseren van een doorlopend, automatisch onderzoeksproces. Het gebruik van de plugin voor onze klanten is gratis. Uniek in onze aanpak is de mogelijkheid om zelfstandig vragenlijsten te formuleren, deze via SMS te bezorgen en de resultaten in dashboards te rapporteren.

Stappenplan - KlantgerichtheidsMonitor:
-	Maak bestand
-	Importeer adressen
-	Verstuur uitnodigingen
-	Deel resultaten

Feedback van klanten te realiseren via
-	Mobiel (sms)
-	E-mail
-	Telefonisch (persoonlijk)

Denk aan reviews op: de website van je bedrijf, de bedrijfspagina op Facebook, of vergelijkingssites, zoals Springest voor het vergelijken van trainingen en opleidingen.

Binnen 24 uur ontvang je onze offerte en binnen 5 werkdagen implementeren wij de KlantgerichtheidsMonitor.

In de installatie handleiding wordt uitgelegd hoe u deze plugin kunt gebruiken. U leert hoe u de widget kunt instellen zodat deze volledig te integreren is in uw eigen website.

Om te starten heb je nodig:

Een licentie op het BusinessMonitor platform, kijk voor informatie over het product op [KlantgerichtheidsMonitor](https://businessmonitor.nl/Oplossingen-KlantgerichtheidsMonitor.html)

1.	Een overzicht (csv of excel) om (handmatig) te kunnen uploaden met daarin:
    -	De naam van betreffende klant
    -	Zijn/haar e-mail adres
    -	Zijn/haar mobiele telefoonnummer
    -	De bedrijfsnaam van de klant
2.	De applicatie stuurt de vragenlijst uit. Deze kunnen door de klant beantwoord worden op alle devices.
3.	De response komt realtime in je dashboard & WordPress Pagina terecht

== Installation ==

EN:
1. Upload the plugin files to the `/wp-content/plugins/businessmonitor` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the `Plugins` screen in WordPress
3. Use the Settings->BusinessMonitor screen to configure the plugin

NL:
1. Upload de pluginbestanden naar `/wp-content/plugins/businessmonitor` of installeer de plugin direct via het Wordpress plugin scherm
2. Activeer de plugin via het `Plugins` scherm in WordPress
3. Gebruik het scherm Instellingen - BusinessMonitor om de plugin te configureren
